## 🚀 How to use your new tools

Once you activate the add-ons, you can access them within the 3D viewport.

### Using the BoxCutter tool
Press the W key to toggle the BoxCutter mode. Click and drag in the viewport to draw shapes on your model. The add-on will automatically cut these shapes into the surface. You can change the shape of the cut by using the menu that appears on the screen.

### Using the Hard-Ops tool
Press the Q key to open the Hard-Ops menu. This menu displays quick actions based on the object you currently have selected. Use this for common tasks like adding bevels, applying operations, or smoothing surfaces.

## 💡 Troubleshooting common issues

If you encounter trouble, check these common fixes first.

*   **Tool not visible:** Ensure you checked the box in the Blender Preferences menu. Restart Blender if the tools do not appear after activation.
*   **Shortcut keys not working:** Blender might have conflicting key assignments. Check the Keymap section in Preferences if the Q or W keys perform other tasks.
*   **Performance lag:** If your viewport slows down, turn off the view-port subdivision settings on your objects. High-poly models require significant computer resources.
*   **File errors:** If the installation fails, ensure you downloaded the full ZIP folder and that the Blender version matches the requirements.
